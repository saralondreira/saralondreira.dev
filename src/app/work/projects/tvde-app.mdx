## Overview

An application built to help TVDE (Transporte Individual e Remunerado de Passageiros em Veículos Descaracterizados a Partir de Plataforma Eletrónica) drivers manage their daily trips, earnings, and vehicle maintenance schedules.

## Technologies Used

- **Mobile Development:** Android
- **Programming:** Java / C
- **Database:** SQL (SQLite for local storage)

## Key Features

- Trip logging and daily earnings dashboard.
- Expense tracking and vehicle maintenance reminders.
- Intuitive user interface tailored for mobile use on the go.
