## Overview

A robust application developed to manage hospital operations efficiently. This project focuses on handling patient records, structuring data, and applying solid Object-Oriented Programming principles.

## Technologies Used

- **Programming:** Java
- **Database:** SQL / Relational Databases
- **Version Control:** Git

## Key Features

- Patient registration and medical history tracking.
- Secure data management and retrieval.
- System architecture designed for scalability and reliability.
